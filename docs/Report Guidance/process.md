# Process 

The Process Summary is found under **Report Sections** → **Process-Summary.tex** in the LaTeX sidebar. It gives the reader a clear picture of what the facility actually makes.

## What to Write

Write **1–2 paragraphs** covering the main steps of the manufacturing process from raw material to finished product. Include any notable sub-processes, equipment, or operational features relevant to energy use. The following figures must be referenced in the text:

- The process flow diagram 
- The facility layout

Draw information from the pre-assessment meeting, the on-site walkthrough, documentation provided by the facility, and personal research on the facility.

## Creating the Figures in PowerPoint

Both figures are created in PowerPoint. Open a PowerPoint presentation in the **Teams folder** for the assesment, and then open the presentation directly in the **desktop application** (not the web version, because the web version does not allow you to change the color of the SmartArt). Create a new slide for each figure.

**Note**
- All text must be **11 pt Times New Roman**
- Everything must fit within the existing slide — do not change the slide dimensions
- To export: press **Ctrl + A** to select everything, then right-click and choose **Save as Picture**. DO NOT TAKE A **SCREENSHOT**.

### Manufacturing Process Flow Diagram

The process diagram is a horizontal sequence of colored boxes connected by arrows, showing each major step in the facility's process. Typically 4–5 steps, but use as many or as few as the process requires.

**To create the diagram:**

1. Go to **Insert → SmartArt**
2. Under the **Process** tab, select the **first option** (basic boxes-and-arrows layout)
3. Add or remove boxes to match the number of steps in this facility's process
4. Label each box with the step name in **11 pt Times New Roman**
5. Color the boxes in order using the color sequence below

**To recolor a box:**
Right-click the box → **Format Shape** → **Fill** → **Color** → **More Colors** → select the **Eyedropper** → navigate back to this page and click the correct color swatch from the image above.

### Facility Layout

The facility layout gives the reader a spatial sense of how the plant is organized — which areas house which operations.

**To create the layout:**

1. On a new slide, paste in an **aerial view of the facility** (from Google Earth or another source)
2. Draw labeled boxes over the image, each covering a distinct zone or area of the building (e.g., production floor, compressor room, warehouse, office)
3. Use the **same color sequence** as the process diagram above (largest to smallest area)

4. Label each box in **11 pt Times New Roman**

If a facility map or floor plan is available from the client, that can be used in place of the aerial photo.
